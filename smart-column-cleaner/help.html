Help & FAQ

Tools

  • De-Duplicate Cells (by Column): select one column → removes/marks duplicates based on mode.
  • Clear Duplicates Within Each Row: select 2+ adjacent columns.
  • Trim Whitespace: removes leading/trailing spaces.
  • Change Case: lowercase / UPPERCASE / Proper Case.
  • Remove Non-ASCII: removes emojis and non-ASCII characters.

Duplicate modes

  • blank – set duplicates to empty.
  • highlight – keep values and mark duplicates pink.
  • shift-cell – remove duplicates and shift values up/left to close gaps.

Quota & Pro

Free tier allows cleaning up to 300 cells/day. Pro users have no daily limit.

Troubleshooting

  • “Select exactly one column” → your selection must be a single column.
  • “No duplicates found” → try removing extra spaces first using Trim.
  • Connection lost → close and reopen the sidebar.
